# navimow-i105-ha

Schlanke Home-Assistant-Custom-Integration für den **Segway Navimow i105 (H-Serie)**.

Reines HTTP-Polling (kein MQTT), `lawn_mower`-Entity (Start/Pause/Dock) + Akku-/Status-Sensoren.

**Status:** Design-Phase. Siehe [docs/2026-05-31-navimow-simple-design.md](docs/2026-05-31-navimow-simple-design.md).

> Abgeleitete Ideen (OAuth-Flow) aus [segwaynavimow/NavimowHA](https://github.com/segwaynavimow/NavimowHA), GPL-3.0.
